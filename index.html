〜「Web コンテンツ作成」の結果〜

本テーマについて

本授業のテーマについて、Webコンテンツ作成におけるHTML、JavaScript、CSSの3つを活用し、これら全てを使って、Webコンテンツのページの装飾や、イベント処理を学び、より実践的な能力を学習する。

プロジェクト課題について

今回のプロジェクト課題では八王子近辺のグルメ情報に関して表示するWebコンテンツを作りました。今回の課題では、CSSはもちろんのこと、JavaScriptを使用して通信なども行いながら、Webコンテンツを作成たわけですが、一番苦労したのは、CSSやJavaScriptを使いながら、Webコンテンツの編集やイベント処理を作りましたが、なかなか、思い通りにはならず、一番時間がかかった部分でもありました。また、内容の見やすさも、考えながら、編集できたら良いなと感じました。

プロジェクト課題で作成したページ:project.html

数当てゲーム

数当てゲームでは、1〜10の数をユーザーが入力し正解よりも大きければ「まちがい。答えはもっと大きいですよ」と表示し、小さければ「まちがい。答えはもっと小さいですよ」と表示する。またユーザーが正解を入力すれば、「正解です．おめでとう!」と表示するようになっている。とてもシンプルなゲームである。第3講にて数当てゲームのユーザーが予測する数値を固定しながら動作を確認し、第4講で表示先をコンソールではなくHTML上に変更し、予測数値を固定しながらも、表示できるように、第5講でイベント処理を追加することで、ユーザーが自由に予想値を入力し、数当てゲームとして動作するようになった。また、各講義時は初めて触るものでもあったため思い通りに行かなかったが、最終的には資料や掲載してある参考資料を復習したことで完成した。

数当てゲームのページ:kazuate.html

各授業回の学習事項

第1回WWWとHTML

今回学習した事柄として、WWWとHTMLについて学びました。最初の導入としてHTMLの構文の学習、また要素についても学び、＜p＞や＜a＞、など実際に使いながら要素を学んでいきました。最後にgithubの使い方を復習しました。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回CSSによるWeb ページの装飾

今回の第2講ではCSSを使ってWebコンテンツの見た目を変更することをメインとして行いました。HTMLの中にある要素の見た目を変えるのはパターンが多くどのようにデザインしていくかはなかなか難しいと感じました。

演習の成果:練習問題2-3:ウェブブラウザ説明ページの装飾

第3回JavaScript の基本

今回では前回のCSSのようなWebコンテンツの編集というものではなくJava scriptを使用して、コンソールにデバックを表示することがメインになりました。また、JavaScriptのプログラミング自体もJavaとは違うものだったので、プログラムを組むのに少し苦戦しましたが、これからJavaScriptを使っていきたいと思います。

演習の成果:練習3-6: オブジェクトの参照

第4回DOM と JavaScript

今回はJavaScriptとDOMを利用して、Webコンテンツの内容の編集や処理を行いました。まだ、JavaScriptを使うのに、慣れないので、projectの編集と同時に慣れていきたと思います。また、HTMLを直接編集するのではなくDOMでの編集をすることで、容易にWebコンテンツの編集ができるので、使っていきたいと思いました。

演習の成果:練習問題4-2から4-5-DOMプログラミング

第5回イベント処理

今回はJavaScriptを使ってイベントの処理を行いました。すでにあるものを消したり、追記したりしました。また、ボタンを作ってイベントの処理を行ったり、キーボードを使ってイベントの処理を行いました。

演習の成果:練習問題5-3:イベントを選択する

  • 工夫点

    • 工夫点としては、project課題では、グルメ情報を掲載する上で、見やすいように意識をし、配色が多すぎると視覚情報が多く、目へのダメージが大きくなってしまうと思ったので、2色に絞り、優しい色を使いました。また、枠やフォントを変更し、レトロゲームのような雰囲気をテーマとして、文字の見やすさやコンテンツとして楽しめるように工夫しました。

  • 苦心点

    • 苦心点としては、JavaScriptやCSSは自分自身初めて触れるもので、プログラムの組み方や、HTMLの仕様を考えて変更するのに、とても苦労しました。枠組みを用意したり、配色を考えたりとうまく行かないところが多くありました。また、講義の後半になるにつれてイベントの処理や、表示の編集が高度になっていき、苦戦するところもありましたが、授業資料に掲載してある参考資料で、やりたいことを表現できました。

  • 強調点

    • 強調点としては、project課題の表示として、ゲーム画面のような雰囲気を出せるようにしました。また、テキストの配置として、ゲーム画面の掲示板のような画面を想像させる表示を作り、お店を探すのに見やすいようにしました。そして、配色の点でも見やすいように2色にこだわり、薄めの色を使用することで、黒が目立つようにして文字を見やすいようにしました。

  • 感想

    • 今回のWebコンテンツの作成実験を通じて、HTML、JavaScript、CSS、DOMの連携の重要性と複雑さを実感しました。HTMLで基本的な構造を作り、CSSでデザインを整え、JavaScriptとDOMで動的な機能を追加する段階としては、Web開発の全体像を理解する上で非常に学びになりました。特にJavaScriptのプログラミングはとても苦労しました。数当てゲームはもちろん、project課題においても、JavaScriptの部分において苦戦しながら課題に取り組んでいました。苦心点でもお話ししましたが、初めて触るプログラムでもあったため、プログラムを組むのにとても時間がかかりました。授業時間の課題は進められましたが、自由表現となるとなかなか進まず、自分のやりたいことがどのようにしたら表現できるのかを考えプログラムとして動作するのか見当がつかず、JavaScriptを動かす上で一番苦労したことでした。しかし、CSSの部分で、Webコンテンツのデザインを変更していく部分は特に取り組みやすく、自分の表現したいことを出せました。今回の結果として、Webコンテンツの作成を通して新たなプログラミング言語の習得から基礎から応用までの実践的な学びができたので、この経験は大きく、Webコンテンツの複雑さを深く理解する良い学びとなりました。
